## Suggested Keywords for LottieFiles Search

- **Tier 0 (First Mover)**: crown, gold crown, king, royalty, vip
- **Tier 1 (OG Founder)**: diamond, gem, crystal, jewel, precious
- **Tier 2 (Alpha Insider)**: rocket, launch, spaceship, blast off
- **Tier 3 (Beta Crew)**: lightning, bolt, thunder, energy, electric
- **Tier 4 (Early Adopter)**: star, sparkle, shine, twinkle, glow
- **Tier 5 (Pioneer Wave)**: target, bullseye, aim, goal, precision
- **Tier 6 (Community)**: bear, teddy bear, cute bear, panda
- **Tier 7 (Default)**: phone, mobile, smartphone, device

## File Format

- Format: JSON (Lottie JSON format)
- Recommended size: < 500KB per file
- Loop: Should be designed to loop seamlessly
- Colors: Will look best with honey/amber tones or white on dark

## Usage

Once files are placed here, the `TierIcon` component will automatically load them.
If a file is not found, the component falls back to the emoji.
